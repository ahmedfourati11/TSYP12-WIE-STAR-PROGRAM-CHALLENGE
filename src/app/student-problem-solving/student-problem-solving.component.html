Problem 1: Reversing a string

Description: This problem asks the user to input a string and then displays the reversed version of that string.

Task:
1. Prompt the user to enter a string.
2. Display the reversed string.

Example Input and Output:
• Input: Python
• Output: The reversed string is: nohtyP

Problem 2: Calculate the Factorial of a Number

Description: This problem requires the user to input an integer, and the program calculates its factorial.

Task:
1. Ask the user to input an integer.
2. Calculate and display the factorial of the number.

Example Input and Output:
• Input: 5
• Output: The factorial of 5 is: 120

Problem 3: IEEE Bookstore

Description: A bookstore sells three types of books:

  • Programming books for 20 dinars each.
  • Math books for 15 dinars each.
  • Electronics books for 30 dinars each.

NB : If the user is an IEEE member, they receive a 10% discount on their total purchase.

Task:
1. Ask the user if they are an IEEE member (Yes/No).
2. Prompt the user to input the number of books purchased in each category.
3. Calculate the total cost, applying a 10% discount if the user is a member.
4. Display the final amount to be paid.

Example Input and Output:
• Input :
• IEEE Member: Yes
• Programming Books: 2
• Math Books: 1
• Electronics Books: 1
• Output: The total amount to be paid is: 76.50 dinars

Problem 4: Find the First Four Perfect Numbers

Description: A perfect number is an integer that equals the sum of its proper divisors (excluding itself). For example, 6 is a perfect number because its divisors are 1, 2, and 3, and their sum is 6.

Task:
1. Identify the proper divisors of a number.
2. Check if the sum of the divisors equals the number itself.

Example Output:
• The first four perfect numbers are: 6, 28, 496, 8128
